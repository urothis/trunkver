# trunkver-rs

## Overview
A library for baking trunkver into your Rust projects

## Usage
```
cargo add trunkver
```

## Example
```
use trunkver::Trunkver;

fn main() {
    println!("{}", TRUNKWHY);
    println!("{}", TRUNKHOW);
    println!("{}", TRUNKWHAT);
    println!("Version without v: {}", TRUNKER);
    println!("Version: {}", TRUNKVER);
}
```
